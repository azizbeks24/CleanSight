# CleanSight

CleanSight is a Streamlit-based data cleaning and analysis application that allows users to upload datasets, preprocess data, and generate insights through an interactive workflow.

## Features

- Upload datasets (CSV, Excel, JSON)
- View dataset summary (rows, columns, types)
- Detect and handle missing values
- Remove duplicates
- Apply transformations (forward fill, backward fill, etc.)
- Generate visualizations (line charts, etc.)
- Export cleaned data

## Workflow

Upload → Clean → Visualize → Export

## Technologies Used

- Python
- Streamlit
- Pandas
